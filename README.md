# Member Data Project

## Features

* Display member data
* Add new member data

## Technologies Used

* Native PHP
* MySQL
* XAMPP

## How to Run

1. Clone this repository
2. Move the project folder to `htdocs`
3. Import `db_quiz_koneksi.sql` into phpMyAdmin
4. Run the project at:

```text
http://localhost/project-anggota
```
